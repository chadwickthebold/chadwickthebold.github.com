Projects

Allsongsmap 2014-2015

I'm really passionate about live music. Most of my taste for the last few years has been informed by the NPR All Songs Considered podcast. I'm working to integrate the NPR story API with the Songkick event calendar to make sure that I don't miss any great shows that are coming to my area. This ongoing project combines a Django-based REST API with a BackboneJS frontend to allow users to search across all artists featured over the entire 13 years of the ASC podcast history and see if any of them are coming to their backyard. A full technical writeup will be available when I finish it!
